# 🚀 极简个人博客系统

基于 Node.js + Express + SQLite 打造的轻量级响应式博客。自带后台管理面板，支持文章状态一键无缝切换！

## 🛠️ 运行指南

1. 确保电脑已安装 Node.js 运行环境。
2. 下载此仓库代码并解压到本地。
3. 在文件夹根目录打开终端，执行以下命令装载依赖：
    npm install
4. 点火启动服务器：
    npm start
5. 打开浏览器，开始访问：
   - 🌍 博客前台：http://localhost:3000
   - ⚙️ 管理后台：http://localhost:3000/admin (默认管理密码请在 config.js 文件中自行修改)
